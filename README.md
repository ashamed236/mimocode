# 设计+前端开发作品集

一个展示设计能力和前端开发技能的个人作品集网站，定位为"设计+前端开发"复合型自由职业者的在线展示平台。

## 项目概述

本项目是一个现代化的个人作品集网站，集成了服务展示、定价方案、项目案例和联系方式等完整功能模块。网站采用响应式设计，完美适配各种设备，展现了专业的设计能力和前端开发技术。

## 功能特点

### 核心功能
- **响应式设计**: 完美适配桌面、平板、手机等各种设备
- **服务展示**: UI/UX设计、响应式网页开发、着陆页设计、企业官网
- **定价方案**: 基础套餐(¥1,500)、专业套餐(¥3,000)、企业套餐(¥5,000+)
- **项目案例**: 展示设计+前端开发的复合项目
- **多渠道联系**: 邮箱、电话/微信、GitHub、接单平台

### 技术特点
- 现代化UI/UX设计
- 平滑滚动和动画效果
- CSS Grid和Flexbox布局
- 原生JavaScript交互
- 语义化HTML5标签

## 技术栈

- **HTML5**: 语义化标签，结构清晰
- **CSS3**: 现代布局（Flexbox、Grid），CSS变量，动画效果

## 设计理念

- **用户体验优先**: 清晰的导航结构，直观的信息层级
- **专业与亲和力并重**: 专业的设计展现能力，友好的沟通方式

## 部署说明

### 1. 本地预览
直接在浏览器中打开 `index.html` 文件即可预览。

### 2. 部署到GitHub Pages
1. 在GitHub上创建新仓库
2. 将代码推送到仓库
3. 进入仓库设置 → Pages
4. 选择 `main` 分支作为源
5. 保存后即可访问

### 3. 部署到其他平台
- **Netlify**: 直接拖拽文件夹到Netlify
- **Vercel**: 连接GitHub仓库自动部署
- **自己的服务器**: 上传文件到Web服务器目录
