# 设计+前端开发作品集

这是一个展示设计能力和前端开发技能的个人作品集网站。

## 功能特点

- 响应式设计，支持各种设备
- 现代化UI/UX设计
- 平滑滚动和动画效果
- 服务和定价展示
- 多平台联系方式

## 部署说明

### 1. 本地预览
直接在浏览器中打开 `index.html` 文件即可预览。

### 2. 部署到GitHub Pages
1. 在GitHub上创建新仓库
2. 将代码推送到仓库
3. 进入仓库设置 → Pages
4. 选择 `main` 分支作为源
5. 保存后即可访问

### 3. 部署到其他平台
- **Netlify**: 直接拖拽文件夹到Netlify
- **Vercel**: 连接GitHub仓库自动部署
- **自己的服务器**: 上传文件到Web服务器目录
