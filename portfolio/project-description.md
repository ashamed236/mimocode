# 设计+前端开发作品集网站

## 项目概述

这是一个展示设计能力和前端开发技能的个人作品集网站，定位为"设计+前端开发"复合型自由职业者的在线展示平台。网站采用现代化设计语言，集成了服务展示、定价方案、项目案例和联系方式等完整功能模块。

## 核心功能

### 1. 响应式设计
- 完美适配桌面、平板、手机等各种设备
- 移动优先的设计理念
- 流畅的跨平台浏览体验

### 2. 服务展示模块
- **UI/UX设计**: 用户界面和用户体验设计
- **响应式网页开发**: 将设计稿转化为高质量网页
- **着陆页设计开发**: 高转化率的产品着陆页
- **企业官网**: 完整的企业官网解决方案

### 3. 定价方案系统
- **基础套餐 (¥1,500)**: 单页面设计，响应式布局，基础交互效果
- **专业套餐 (¥3,000)**: 多页面网站，UI/UX设计，交互动画，SEO优化
- **企业套餐 (¥5,000+)**: 完整官网设计开发，后台管理系统，1年维护

### 4. 项目案例展示
- 科技公司官网设计与开发
- 产品发布着陆页
- 精品电商界面设计

### 5. 多渠道联系
- 邮箱联系
- 电话/微信联系
- GitHub展示
- 接单平台链接（猪八戒网、一品威客、Upwork、Fiverr）

## 技术栈

### 前端技术
- **HTML5**: 语义化标签，结构清晰
- **CSS3**: 现代布局（Flexbox、Grid），CSS变量，动画效果
- **JavaScript**: 原生JS实现交互功能

### 设计特点
- **色彩方案**: 以蓝色(#2563eb)为主色调，金色(#fbbf24)为点缀色
- **字体系统**: 系统字体栈，确保最佳可读性
- **视觉层次**: 清晰的排版层次，突出重点信息
- **微交互**: 悬停效果、平滑滚动、动态导航栏
